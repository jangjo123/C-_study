오늘의 주제: VI 에디터

vi
 - 빈파일 열림 (새문서)

vi [파일명]
 - 존재하는 파일이면 파일 열림
 - 빈 파일 열림 (새문서)

모드형 편집기
- vi : 입력 모드 / 명령 모드
  명령모드 -> 입력모드(i)
  입력모드 -> 명령모드 (ESC)

파일 저장/종료
 q(quit) !(강제) w(write)
:q 그냥 종료
:q! 그냥 종료 (고친 사항이 있어도 강제 종료)
:w [파일명] 파일 저장
:wq 파일 저장하고 종료
:wq! 파일 저장하고 종료 (고친 사항이 있어도 강제 종료)

명령모두 -> 입력모드
- i (현재 커서 위치)
- a (커서 뒤)
- o (커서 기준 다음 행)
- I (커서 기준 행의 시작)
- A (커서 기준 행의 끝)
- O (커서 기준 이전 행)

화면 이동
 - 초기 vi에서는 상하좌우(kjhl)
 - 통상적인 키로도 매핑이 되어있음
   상하좌우 방향키
   [home] [end] 행의 앞/끝 이동
   화면 위/아래 [page up][page down]

특정 행 이동
 G or :$ (파일의 마지막 행으로 이동)
행번호G or :행번호 (특정 행 번호 이동)
set nu(number) : 행 번호 표시
set nonu : 행 번호 숨김

수정
- r (커서의 글자를 다른 글자로 수정)
- s (커서의 글자 삭제, ESC 입력할때까지 입력)
- #cw (change word) 단어 삭제하고, 해당 위치에 입력 모드
- #cc (change change) 행을 삭제하고, 해당 위치에 입력 모드
- C (change) 커서 위치부터 행의 끝까지 수정


삭제
 - #x 커서 위치 글자 삭제
- #dw (delete word) 커서 위치 단어 삭제
- #dd (delete delete) 커서 위치 행 삭제
- D (Delete) 커서 위치부터 행 끝까지 삭제

명령 취소
- Ctrl + z (fg로 돌아간다)
- u(undo) 명령 취소
- U(undo) 행에서 한 모든 명령 취소
- :e! (erase) 마지막 저장 내용 이후의 변경내용 다 버림

복붙
- #yy (copy copy) 커서 위치 행 복사
- p (paste) 커서 위치한 행 아래에 붙임
- p (paste) 커서 위치한 행 위에 붙임
- cc dd 상태에서 p 잘라 붙이기 가능

검색
- /단어 단어 검색(아래 방향으로 검색)
- ?단어 단어 검색(윗 방향으로 검색)
- n (검색 방향 : 다음)
- N (Next 검색 방향: 이전)

바꾸기
- :[범위]s/단어1/단어2  범위에서 첫번째 단어1 -> 단어2
- :[범위]s/단어1/단어2/g 범위에서 단어1 -> 단어2

  :s/단어1/단어2      커서 행에서 나오는 첫번째 단어1->단어2
  :s/단어1/단어2/g   커서 행에서 나오는 단어1-> 단어2
  :%s/단어1/단어2/g  파일 전체에서 단어1->단어2
  :5,10s/단어1/단어2/g 5~10줄 단어1->단어2

기타
. 이전 명령 반복
~ 위치의 글자 대/소 변경
